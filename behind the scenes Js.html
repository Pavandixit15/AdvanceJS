:- So javascript is Always Hosted in Environment and that is Most typically browser such as Chrome , firefox , safari etc , This is Where javascript Runs ,

:- there can also be other hosts such as Node js web server or even some Applications that accept javascript code input

:- so we will concertrate on browser so when we Write JS code and want to run the code there is lot of stuff happening behind the scenes ,So what happens is that the host where JS is hosted it has somekind of Javascript Engine that takes our code and executes it

:-The Javascript engine is a Program which executes Javascript code , There are Many different Engines are there like googles V8 engine and many more

:- The first Thing that happens inside the Engine is that our code is Parsed by Parser which basically reads Our code line by line and checks if the syntax of the code that we gave it is correct , that is means the Parser know the Javascript rules & how it has to be written in order to be correct

and if we make a mistake it will throw a error and stops the execution if eerything is correct though then the Parser Produces a data structure known as a abstract syntax tree which is then translated to machine code which is executed directly by the computers processors and it only when our code already converted to a machine code actually runs and does it work
